# Blender 3D模型项目文档

## 项目概述

本项目包含两个Blender Python程序化生成的3D模型：
- **天安门城楼** — 中国标志性建筑
- **圆头耄耋（橘猫）** — 网络热梗角色

---

## 一、天安门城楼模型

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 文件名 | `create_tiananmen.py` |
| 输出文件 | `tiananmen.blend`（174KB） |
| 生成方式 | Blender Python API 程序化建模 |
| 渲染引擎 | Cycles |
| 分辨率 | 1920×1080 |

### 1.2 模型结构

```
天安门城楼
├── 城墙基础（含城门洞布尔运算）
│   └── 城墙垛口 × 22
├── 城楼主体
│   ├── 红色立柱 × 18
│   ├── 窗户 × 14
│   ├── 匾额 × 2
│   └── 城楼平台
├── 屋顶结构
│   ├── 下层屋顶（重檐歇山顶）
│   ├── 上层屋顶
│   ├── 正脊
│   └── 垂脊 × 5
├── 装饰元素
│   ├── 华表 × 2（柱身+柱头+柱座）
│   ├── 石狮 × 2
│   └── 标语牌 × 2
├── 金水桥
│   ├── 桥面
│   └── 栏杆 × 14
└── 场景设置
    ├── 地面
    ├── 主光源（太阳光）
    ├── 补光（面光源）
    └── 相机
```

### 1.3 材质系统

| 材质名称 | 颜色值 | 用途 |
|---------|--------|------|
| 城墙红 | (0.8, 0.1, 0.1) | 城墙、城楼主体 |
| 屋顶黄 | (0.9, 0.8, 0.2) | 屋顶、屋脊、匾额 |
| 城砖灰 | (0.5, 0.5, 0.5) | 地面、石狮 |
| 汉白玉 | (0.95, 0.95, 0.95) | 华表、桥栏杆、标语牌 |
| 窗户 | (0.2, 0.15, 0.1) | 窗户木框 |

### 1.4 关键尺寸参数

```python
城墙基础: scale = (12, 3, 2)    # 长12 宽3 高2
城楼主体: scale = (10, 2, 4)    # 长10 宽2 高4
城门洞: 宽2.5, 高2
下层屋顶: radius1=3, depth=2
上层屋顶: radius1=2.5, depth=1.5
华表高度: 6单位
```

### 1.5 历史准确度说明

模型为**简化艺术版本**，主要用于学习和展示：
- 城楼比例大致准确（重檐歇山顶）
- 省略了部分精细雕刻细节
- 华表和石狮子为简化造型
- 省略了毛主席画像等政治元素

## 二、圆头耄耋（橘猫）模型

### 2.1 基本信息

| 项目 | 内容 |
|------|------|
| 文件名 | `maodie_cat.py` |
| 输出文件 | `maodie_cat.blend`（205KB） |
| 生成方式 | Blender Python API 程序化建模 |
| 渲染引擎 | Cycles |
| 分辨率 | 1920×1080 |

### 2.2 角色背景

**圆头耄耋**是2025年走红的网络热梗：
- **来源**："圆头猫爹"的谐音演变
- **原型**：博主"白手套&马犬旺财"拍摄的流浪橘猫
- **特征**：脑袋圆、脾气差、攻击性强
- **行为**：哈气、飞机耳、脊背龙、喂不熟

### 2.3 模型结构

```
圆头耄耋（橘猫）
├── 身体
│   ├── 身体主体（胖椭圆）
│   └── 肚子（浅色区域）
├── 头部
│   ├── 头部主体（圆头）
│   ├── 脸部浅色区域
│   ├── 左耳（飞机耳 -60°）
│   │   └── 左耳内侧（粉色）
│   ├── 右耳（飞机耳 60°）
│   │   └── 右耳内侧（粉色）
│   ├── 眉弓 × 2（皱眉效果）
│   └── 额头M纹（虎斑特征）
├── 眼睛
│   ├── 左眼
│   │   ├── 眼白
│   │   ├── 虹膜（黄绿色）
│   │   └── 瞳孔（竖瞳）
│   └── 右眼
│       ├── 眼白
│       ├── 虹膜（黄绿色）
│       └── 瞳孔（竖瞳）
├── 嘴巴（哈气状态）
│   ├── 上颚
│   ├── 下颚（张嘴）
│   ├── 口腔内部（红色）
│   ├── 舌头（粉色）
│   ├── 左犬齿
│   └── 右犬齿
├── 鼻子
│   ├── 鼻头（粉色三角）
│   └── 鼻梁纹
├── 胡须 × 6（左右各3）
├── 四肢
│   ├── 左前腿 + 爪 + 肉垫
│   ├── 右前腿 + 爪 + 肉垫
│   ├── 左后腿 + 爪
│   └── 右后腿 + 爪
├── 尾巴
│   ├── 尾巴分段 × 6（带条纹）
│   └── 尾巴尖
└── 场景设置
    ├── 地面
    ├── 主光源（太阳光）
    ├── 补光（面光源）
    └── 相机
```

### 2.4 材质系统

| 材质名称 | 颜色值 (RGBA) | 用途 |
|---------|---------------|------|
| 橘猫橙 | (0.95, 0.55, 0.15, 1.0) | 主体毛色 |
| 腹部浅色 | (0.98, 0.85, 0.65, 1.0) | 肚子、脸部 |
| 鼻头粉 | (0.95, 0.6, 0.65, 1.0) | 鼻头、肉垫、舌头 |
| 猫眼黄绿 | (0.6, 0.8, 0.2, 1.0) | 虹膜 |
| 瞳孔黑 | (0.02, 0.02, 0.02, 1.0) | 瞳孔 |
| 眼白 | (0.95, 0.95, 0.92, 1.0) | 眼白、犬齿 |
| 口腔 | (0.8, 0.3, 0.3, 1.0) | 嘴巴内部 |
| 虎斑纹 | (0.7, 0.35, 0.08, 1.0) | 额头M纹、身体条纹 |

### 2.5 特征实现细节

| 特征 | 实现方式 | 参数 |
|------|---------|------|
| 圆头 | 头部球体横向拉伸 | scale=(1.05, 1.0, 0.95) |
| 飞机耳 | 锥体向外旋转60° | rotation_euler.y = -60° |
| 哈气 | 上下颚分离 + 口腔可见 | 上颚z=2.1, 下颚z=2.0 |
| 竖瞳 | 瞳孔纵向拉伸 | scale=(0.4, 0.5, 1.0) |
| 凶狠眼神 | 眉弓向内倾斜 | rotation_euler.z = ±15° |
| 虎斑纹 | 深色材质条纹 | 额头M纹 + 尾巴环纹 |
| 粗壮体型 | 身体椭圆加宽 | body.scale=(1.1, 0.85, 0.95) |

### 2.6 性格表现对照

| 真实行为 | 模型表现 |
|---------|---------|
| 飞机耳 | 耳朵向外平躺 |
| 哈气 | 张嘴露齿 |
| 攻击姿态 | 皱眉 + 竖瞳 |
| 脊背龙 | （需动画实现） |

## 三、技术实现对比

| 对比项 | 天安门城楼 | 圆头耄耋 |
|--------|-----------|---------|
| 建模类型 | 建筑硬表面 | 生物有机体 |
| 主要几何体 | Cube + Cone | Sphere + Cone |
| 复杂度 | 中等（结构规整） | 较高（有机形态） |
| 材质数量 | 5种 | 8种 |
| 布尔运算 | 有（城门洞） | 无 |
| 细分曲面 | 无 | 有（Subdivision） |
| 动画就绪 | 否 | 骨骼绑定后可动画 |
| 多边形数 | ~5K | ~8K |

## 四、使用说明

### 4.1 运行脚本

```bash
# 生成天安门模型
"C:\Program Files\Blender Foundation\Blender 5.1\blender.exe" --background --python create_tiananmen.py

# 生成耄耋橘猫模型
"C:\Program Files\Blender Foundation\Blender 5.1\blender.exe" --background --python maodie_cat.py
```

### 4.2 在Blender中打开

```bash
# 打开天安门
"C:\Program Files\Blender Foundation\Blender 5.1\blender.exe" tiananmen.blend

# 打开耄耋橘猫
"C:\Program Files\Blender Foundation\Blender 5.1\blender.exe" maodie_cat.blend
```

### 4.3 渲染

在Blender中按 `F12` 渲染当前视图。
