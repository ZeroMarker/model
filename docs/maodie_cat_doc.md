## 角色背景

### 梗起源

**圆头耄耋**是2025年走红的网络热梗：
- **来源**："圆头猫爹"的谐音演变
- **原型**：博主"白手套&马犬旺财"拍摄的一只闯入其家中的流浪橘猫
- **含义**：指代脑袋圆、脾气差的小猫形象

### 角色特征

| 特征 | 描述 |
|------|------|
| 品种 | 橘猫（橙色虎斑） |
| 头型 | 圆润饱满（耳朵紧贴时正面呈圆形） |
| 性格 | 暴躁、攻击性强、喂不熟 |
| 行为 | 哈气、飞机耳、脊背龙、主动攻击 |
| 状态 | 又馋又横 |

## 特征实现对照

| 角色特征 | 实现方式 | 关键参数 |
|---------|---------|----------|
| 圆头 | 球体横向拉伸 | head.scale=(1.05, 1.0, 0.95) |
| 飞机耳 | 锥体向外旋转 | rotation_euler.x=-60°, z=±20° |
| 哈气 | 上下颚分离 | 上颚z=2.1, 下颚z=2.0 |
| 竖瞳 | 瞳孔纵向拉伸 | pupil.scale=(0.4, 0.5, 1.0) |
| 凶狠眼神 | 眉弓向内倾斜 | brow.rotation_euler.z=±15° |
| 虎斑纹 | 深色材质条纹 | 额头M纹 + 尾巴环纹 |
| 粗壮体型 | 身体椭圆加宽 | body.scale=(1.1, 0.85, 0.95) |
| 猫鼻 | 六棱锥体 | Cone(vertices=6) |
| 尖牙 | 细长锥体 | Cone(radius1=0.03, depth=0.1) |

## 网络热梗背景

### 出处

源自B站/抖音博主"白手套&马犬旺财"发布的视频，记录了一只闯入家中的流浪橘猫：
- 该猫具有极强攻击性
- 对投喂者毫不领情
- 动不动就哈气、伸爪
- 被网友戏称"圆头猫爹"

### 传播

- 2025年开始在B站、抖音广泛传播
- "耄耋"谐音"猫爹"成为热门梗
- 衍生出大量表情包、二创内容
- 成为"坏脾气猫"的代名词

### 典型描述

> "从来没有一只猫这么令人讨厌，哈气，脊背龙，飞机耳，喂不熟，主动攻击饲主"
