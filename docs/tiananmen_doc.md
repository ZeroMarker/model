# 天安门城楼3D模型文档

## 基本信息

| 项目 | 内容 |
|------|------|
| 文件名 | `create_tiananmen.py` |
| 输出文件 | `tiananmen.blend`（174KB） |
| 生成方式 | Blender Python API 程序化建模 |
| 渲染引擎 | Cycles |
| 分辨率 | 1920×1080 |

---

## 模型结构

```
天安门城楼
├── 城墙基础（含城门洞布尔运算）
│   └── 城墙垛口 × 22
├── 城楼主体
│   ├── 红色立柱 × 18
│   ├── 窗户 × 14
│   ├── 匾额 × 2
│   └── 城楼平台
├── 屋顶结构
│   ├── 下层屋顶（重檐歇山顶）
│   ├── 上层屋顶
│   ├── 正脊
│   └── 垂脊 × 5
├── 装饰元素
│   ├── 华表 × 2（柱身+柱头+柱座）
│   ├── 石狮 × 2
│   └── 标语牌 × 2
├── 金水桥
│   ├── 桥面
│   └── 栏杆 × 14
└── 场景设置
    ├── 地面
    ├── 主光源（太阳光）
    ├── 补光（面光源）
    └── 相机
```

## 历史背景

天安门城楼始建于明永乐十五年（1417年），原名"承天门"，清顺治八年（1651年）改建后称"天安门"。

**建筑特点：**
- 重檐歇山顶
- 城台开五个券门
- 城楼面阔九间，进深五间
- 60根巨柱支撑
- 屋顶面积约2000平方米

**本模型为简化艺术版本**，主要用于学习和展示，省略了部分精细雕刻和政治元素。
